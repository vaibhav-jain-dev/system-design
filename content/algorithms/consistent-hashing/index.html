{{phase 1 "The Problem: Why Not Modulo Hashing?" "3 min"}} {{say "When distributing keys across N servers, the naive approach is hash(key) % N. The problem: when you add or remove a server, N changes and almost every key remaps to a different server. Consistent hashing solves this by only moving K/N keys when a node changes."}} {{table "Modulo Hashing Failure Modes" (rows (row "Scenario" "Keys Remapped" "Impact") (row "3 servers, add 1" "~75% of keys move" "Cache stampede, massive DB load") (row "10 servers, lose 1" "~90% of keys move" "Near-total cache invalidation") (row "100 servers, add 1" "~99% of keys move" "Catastrophic rebalancing event") (row "Consistent hashing" "Only K/N keys move" "Minimal disruption, no stampede") )}} {{thought "With modulo hashing on 100 cache servers, losing one node remaps 99% of keys. Every key misses cache and hits the database simultaneously. This thundering herd takes down production systems. Consistent hashing limits the blast radius to 1/N of keys."}} {{key "Consistent hashing is the go-to algorithm whenever you distribute data across a cluster that can grow or shrink. Used by DynamoDB, Cassandra, Memcached, Redis Cluster, Akamai CDN, and Discord."}} {{phase 2 "The Hash Ring Concept" "5 min"}} {{say "Consistent hashing maps both servers and keys onto a circular hash space called a ring, ranging from 0 to 2^32. O(log(N * vnodes)) via binary search.""" if not self.ring: return None h = self._hash(key) idx = bisect.bisect_right(self.sorted_keys, h) if idx == len(self.sorted_keys): idx = 0 # wrap around return self.ring[self.sorted_keys[idx]] def get_replicas(self, key: str, count: int = 3) -> list: """Walk clockwise collecting distinct physical nodes.""" if not self.ring: return [] h = self._hash(key) idx = bisect.bisect_right(self.sorted_keys, h) replicas, seen = [], set() for _ in range(len(self.sorted_keys)): if idx == len(self.sorted_keys): idx = 0 node = self.ring[self.sorted_keys[idx]] if node not in seen: replicas.append(node) seen.add(node) if len(replicas) == count: break idx += 1 return replicas # --- Usage --- ring = ConsistentHashRing(["cache-1", "cache-2", "cache-3"]) for key in ["user:1001", "user:1002", "session:abc"]: print("{} -> {} replicas: {}".format( key, ring.get_node(key), ring.get_replicas(key, 2))) ring.add_node("cache-4") # only ~1/4 of keys move ring.remove_node("cache-2") # only its keys move clockwise `}} {{phase 5 "Rebalancing Analysis" "5 min"}} {{say "The key property of consistent hashing is minimal disruption.
