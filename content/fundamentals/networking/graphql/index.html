{{phase 1 "What Is GraphQL?" "3 min"}} {{say `GraphQL is a query language for APIs and a runtime for executing those queries:
  • Created by Facebook in 2012, open-sourced in 2015 — solves the over-fetching and under-fetching problems of REST
  • One endpoint (POST /graphql) replaces dozens of REST endpoints — clients describe exactly what they need
  • Strongly typed schema — all data types defined in SDL (Schema Definition Language); invalid queries rejected before execution
  • 3 operation types: Query (read), Mutation (write), Subscription (real-time push via WebSocket)
  • Used by: GitHub, Twitter, Shopify, Airbnb, Netflix, PayPal, Coursera
`}} {{diagram "fund-graphql-overview"}} {{think "GraphQL shifts the 'what data do I need' question from the server to the client — powerful but requires careful server-side protection."
