{{phase 1 "What Is CAP Theorem?" "3 min"}} {{say "CAP Theorem says a distributed system can guarantee at most two of three properties:
  • Consistency — every read returns the most recent write or an error
  • Availability — every request gets a non-error response
  • Partition Tolerance — the system continues operating despite network splits
In practice, partition tolerance is non-negotiable — so the real decision is CP vs AP."}} {{diagram "fund-cap-overview"}} {{think "CAP is often misunderstood as a three-way tradeoff. It is not. (whyNot "ignoring partition tolerance" "A system that assumes no network failures will catastrophically fail when one occurs. (whatIf "a network split separates two database nodes" "A CP system refuses writes on the minority partition.
